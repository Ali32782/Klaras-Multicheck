# Klaras Multicheck

Trainer: https://ali32782.github.io/Klaras-Multicheck/

## Für Papa (Eltern-Bericht)

1. Klara öffnet den Trainer und trainiert wie gewohnt.
2. Im Trainer: **Statistiken → Eltern-Bericht öffnen** (oder Link / Wochenbericht kopieren).
3. Papa öffnet den Link — **kein Login nötig**.

Direkte URLs:
- Trainer: https://ali32782.github.io/Klaras-Multicheck/
- Eltern-Bericht: https://ali32782.github.io/Klaras-Multicheck/report.html
- Firebase-Dashboard (optional): https://ali32782.github.io/Klaras-Multicheck/dashboard.html

Sessions werden lokal im Browser gespeichert (`localStorage`). Der Trainer versucht zusätzlich, Sessions anonym nach Firebase zu syncen (falls erlaubt).
